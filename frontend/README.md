# LMS Mini App

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

Runs on:

```
http://localhost:5000
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

## Features

- View all courses
- View course lessons
- Mark lessons completed (UI only)
- React + Vite frontend
- Node + Express backend
